AI mountain garden planner

This project uses the SheCodes API to give you a plan for your garden. The page asks you to type in a month. A texts blinks to show you it is generating a plan. Then the SheCodes AI generates a pan in four parts: plan for trees, for ground care, for plants, and for flowers. The project is open-sourced on Github and is hosted on netlify.
